# `macsay`

> Let the Mac say it!

Like [`cowsay`](https://en.wikipedia.org/wiki/Cowsay) but it's the original Macintosh "information" dialog icon instead of a cow.

## Usage

Requires Python 3.6+.

```bash
positional arguments:
  text        Text to say. Can also be stdin.
```
